# Tik Tok Clone

- Next JS, TypeScript
- Styling: Tailwind CSS
- Authentication: Google OAuth
- Icons: React Icons
- State Management: Zustand
- Backend: Sanity

## Functionality

- Fully Responsive
- Upload Video
- Comment and Like Videos
- Search Videos, Video Categories and Usernames
- Suggested Accounts
